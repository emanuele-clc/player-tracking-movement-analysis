Player Tracking & Movement Analysis

Computer-vision pipeline turning real broadcast football video into player tracking data, then clustering movement patterns and generating per-player heatmaps and an off-ball space-creation score.

The pipeline (detection & tracking, pitch calibration, clustering, heatmaps) is being built out — this dashboard will go live with the same interactive style as the companion xG model dashboard once real match data has been processed.

Source and build plan: github.com/emanuele-clc/player-tracking-movement-analysis
